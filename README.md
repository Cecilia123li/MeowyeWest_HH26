# MeowYe West

## Description:
MeowYe is an infinite-runner wild west inspired game where players can play as a siamese cat navigating the wild west. This project utilizes computer vision with Unity 6, allowing players to control MeowYe with simple hand motions in real-time.

## Game System/Environment Requirements:
- OS: Windows 10/11 (64-bit)
- Hardware: A working webcam

## Installation Instructions:
1. Download the .zip file from the latest release.
2. Extract the folder to your preferred location
3. Ensure the webcam is functional
4. Double click the .exe to load the game!

## How to play:
- Position youself so that the camera can capture your hand movements
- Make a handsign to see how MeowYe reacts

## Hand Control Manual
- Stop ✋
- Jump 👆
- Continue (hold the handsign)
- Right ✊ -> then stick your thumb out to the right direction
- Left ✊ -> then stick your thumb out to the left direction
